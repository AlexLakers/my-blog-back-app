# my-blog-back-app
REST-Service блог для публикации и управления постами, с возможностью
оставлять и редактировать комментарии, ставить лайки и т.д. 
В конкретном примере используется в паре с фронтенд-приложением "my-blog-front-app", 
которое позволяет взаимодействовать с бекенд - приложением в форме диалога.
